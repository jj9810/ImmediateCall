# ImmediateCall
an Android application helps call someone immediately

# 설명
**ImmediateCall**은 비상 연락처를 띄워 즉시 연락할 수 있는 잠금 화면입니다.

MIT 라이선스에 따라 배포합니다.

## Packages
`ImmediateCall`의 패키지 목록입니다.

* `lockscreen`: 잠금 화면
* `managecontact`: 연락처 관리 화면
* `receiver`: 리시버
* `util`: 유틸리티
